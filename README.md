# 🐕 Pluto AI Assistant
### Made by Sashank Viswanatham

Pluto is a smart AI assistant that lives on the edge of your screen. Click the golden dog tab and get instant help with anything.

## Features
- 🐕 Always available from the side of your screen
- 🤖 Real AI answers powered by Groq (free!)
- 🎤 Voice input — talk to Pluto
- 🔊 Pluto talks back with Auto-speak
- 🖥️ My Screen — Pluto looks at your screen and helps
- 📎 Upload images and files

## How to use
1. Download the app for Mac or Windows from Releases
2. Open Pluto
3. Go to console.groq.com and get a free API key
4. Paste it in Pluto and you're good to go!

## Built with
- Electron.js
- Groq AI (free!)
- HTML, CSS, JavaScript
